# Task 2: REST API Client

## Description
This task involves creating a Java application that consumes a public REST API,
handles HTTP requests, parses JSON responses, and displays the data in a structured format.

## API Used
- Open-Meteo Weather API (Public API)

## Concepts Used
- HTTPURLConnection
- REST API
- JSON Parsing
- Java I/O

## Outcome
Learned how to integrate external APIs with Java applications.
